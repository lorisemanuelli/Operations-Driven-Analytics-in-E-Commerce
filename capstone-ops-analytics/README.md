# Capstone Ops Analytics

This repository contains a clean, GitHub-ready package for the **Operations-Driven Analytics in E-Commerce** capstone.

## Contents
- `data/` ? placeholder for raw and processed datasets (not included)
- `notebooks/` ? analysis notebooks (add your own)
- `src/` ? forecasting + optimization pipeline modules
- `reports/` ? slides, figures, and exported results

## Quick Start
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Notes
- Add your datasets to `data/raw/`.
- Use `src/train.py` as the training entrypoint (template included).
